Godo.txt ß1.0 — Keyboard Shortcuts & Help

Navigation
↑/↓ Navigate tasks
Shift + ↑/↓ Select multiple tasks
Ctrl + ↑/↓ Move task up/down
( Add new task
Ctrl + x Cut selected task(s)
Ctrl + v Paste cut task(s) at cursor
Task Management
Space
Double click Edit selected task
DEL Delete selected task(s)
x Toggle task completion
Filters & Organization
c Cycle completion state filter
w Cycle due date filter
q Cycle priority filter
0 Clear all filters
f Find task (focus search box)
Priority & Dates
Alt + ↑/↓ Increase/decrease priority
p Set priority for selected task(s)
s Set due date for selected task(s)
d Delay due date for selected task(s)
. Reload tasks from file
Filter Sets
1-9 Load filter set (0 clears all filters)
Ctrl + Alt + 1-9 Save current filters as set
Other
Ctrl + a Select all tasks
Ctrl + c Copy selected tasks to paste into a document
r Sort by priority (file will be saved in this order)
t Sort by due date (file will be saved in this order)
? Show this help window
